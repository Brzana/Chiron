# Chiron

A web application designed to enhance learning through thoughtful integration of generative AI that fosters critical thinking and deeper understanding.

## About

The name "Chiron" honors the legendary centaur from Greek mythology—a wise tutor who guided heroes like Achilles and Heracles. Just as Chiron provided personalized guidance to develop true mastery, this platform leverages AI to create meaningful learning experiences.

## Tech Stack

- **Database**: PostgreSQL with pgvector for vector embeddings
- **Backend**: FastAPI
- **Frontend**: Next.js
